## 📋 Описание
HelpDesk - это веб-приложение для управления заявками. Позволяет создавать, редактировать, удалять и отслеживать статус выполнения заявок.

## 🛠 Технологии
- HTML5
- CSS3
- JavaScript (ES6+)
- Webpack
- Fetch API
- GitHub Actions
